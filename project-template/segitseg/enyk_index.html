Java-s Nyomtatványkitöltő Program Súgó

Tartalomjegyzék

Tárgymutató

Ez a programcsomag számítógéppel támogatott nyomtatványkitöltést megvalósító eszköz. Ezzel a programmal tudja elektronikus nyomtatványát elkészíteni és feladásra megjelölni. Amennyiben Ön internet kapcsolattal rendelkezik, segítséget nyújt az elektronikus feladásban is. Jelen Súgófájl a program használatához ad segítséget.
